## Bu klasör neden var?

`activeContext.md` "şu an ne oluyor" özetini tutar — kısa kalmalı.
`progress.md` "checklist" tutar — durum görünür olmalı.
`fazlar/fazN.md` ise **derin implementation notu**: dosya yapısı, modül-modül açıklama, plana göre verilen ek kararlar, ortam notları, bilinen sınırlamalar.

Bir başka ajan veya sen kendin, aylar sonra geri döndüğünde "Faz 2'de ne yapmıştık?" sorusuna tek dosya cevap versin diye.
